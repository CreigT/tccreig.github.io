# Tc Creig Portfolio

Futuristic high-authority personal portfolio website for **Tc Creig** — AI Systems Architect, Founder of Creignificent LLC, Cybersecurity student, and automation entrepreneur.

## Design Direction
- Cyberpunk command center meets luxury AI laboratory
- Deep space black background (#05070f)
- Elegant gold (#d4af37) + electric cyan (#00e5ff) lighting
- Glassmorphism panels, animated canvases, holographic effects
- Fully responsive mobile-first

## Tech
- Single-page app (index.html + styles.css + script.js)
- Tailwind CSS (Play CDN)
- Pure Canvas + Vanilla JS for all visualizations (no heavy frameworks)
- Three.js not used — everything is lightweight and self-contained

## Sections
- Cinematic loading screen
- Hero with live starfield + drifting particles + shooting stars
- About + Credentials
- 20 animated glowing skill cards
- 10 detailed futuristic project cards + detail modals
- Creignificent LLC business operations
- AI Agent Lab (live terminal + neural network viz + threat intel)
- 6 premium digital product cards + modals
- Futuristic contact form (simulated secure transmission)
- Fully functional keyboard shortcuts

## Quick Start

### Option 1: Direct open
Double-click `index.html` (works offline)

### Option 2: Local server (recommended)
```powershell
# Using Python
python -m http.server 5173

# Or with Node
npx serve .
```

Then open http://localhost:5173

## Keyboard Shortcuts
- `?` — Jump to AI Lab
- `Cmd/Ctrl + K` — Jump to Contact
- Press `c` three times on hero for visual easter egg

## Customization
- Update social links in index.html (LinkedIn, GitHub, etc.)
- Edit projects array in script.js to add/remove case studies
- Change colors via CSS variables in styles.css

## Deployment
Ready for Vercel, Netlify, or GitHub Pages out of the box.
